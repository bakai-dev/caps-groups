## Caps groups test

How to use with Docker:
=========================
Install composer:
```
make init 
```
Run script:
```
make run-app 
```

How to use local PHP CLI:
=========================
Install composer:
```
composer install
```

Run script:
```
php public/index.php 
```

Run test:
=========================
Docker:
```
make run-test
```

Local:
```
php vendor/bin/phpunit
```
